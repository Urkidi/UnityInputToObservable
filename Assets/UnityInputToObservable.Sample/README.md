Unity Input To Observable Sample

## Requirements
- Unity 6000.2 or later
- [R3](https://github.com/Cysharp/R3) (must be installed manually)
- Microsoft.Extensions.DependencyInjection (must be installed Via NuGet  for Unity)
- UnityInputToObservable.DependencyInjection package
